# SuperPadel – Padelio PRO

Vieno failo padelio turnyrų (Americano/Mexicano/Mix) įrankis.

## Deploy į Vercel
1. Įkelk šį repo į GitHub
2. Vercel -> New Project -> Import
3. Domeną (superpadel.lt) pririšk per Vercel Domains.

## Lokaliai
Atidaryk `index.html` naršyklėje arba paleisk paprastą serverį:
```bash
python -m http.server 5173
```
